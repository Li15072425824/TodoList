一、项目说明 & 功能清单（完整版，实用无冗余）
✅ 最终实现的完整功能
新增待办：输入待办内容，回车 / 点击按钮快速添加任务；
状态切换：点击待办✅勾选 / 取消勾选，完成的待办自动划线变灰；
删除待办：单个待办右侧提供删除按钮，点击删除对应任务；
筛选功能：3 个筛选标签「全部 / 已完成 / 未完成」，精准过滤待办列表；
清空全部：一键清空所有待办任务，带二次确认防误触；
数据持久化：基于localStorage本地存储，刷新页面 / 关闭浏览器，待办数据不会丢失；
空数据提示：无待办时展示友好的空页面提示；
响应式适配：兼容 PC 端 / 手机端，页面自适应展示，移动端体验友好。
✅ 技术栈（最优组合，无冗余，前端必备）
plaintext
Vue3 + Vite + Pinia + CSS3(Flex布局) + localStorage + 原生JS
为什么这么选：不用引入任何 UI 库 / 额外依赖，纯原生生态实现，体积小、运行快，同时覆盖前端高频核心知识点，学完直接复用在工作中；
无 TS：降低入门门槛，纯 JS 开发，所有前端开发者都能直接上手，后续可无缝加 TS 改造。
